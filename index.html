A little bit about me...

I am a sports enthusiast with one year of experience in sports tech software engineering and data science. I earned a Bachelor’s degree from Johns Hopkins University in Chemical & Biomolecular Engineering and minored in Applied Mathematics & Statistics with a focus in Data Science. Recently, I built computer vision and analytics models for football and lacrosse—such as a quarterback dropback detector and game-level athleticism metrics using clustering and Bayesian methods. On the engineering side, I developed high‑performance data pipelines and APIs with FastAPI, PostgreSQL, and Polars, and shipped microservices with Docker, Kubernetes, and Redis for wearable‑data integrations.

Basketball Stats App

A work-in-progress personal app to track and compare player statistics. The backend is written in Python using FastAPI for a fast, typed web API, SQLAlchemy for ORM models, and Alembic for database migrations. PostgreSQL is the primary datastore, with Docker and pgAdmin used to run and manage a reliable local environment. The frontend is built with React and TypeScript to enable a type-safe, component-driven UI. The repo also includes backend tests and helper scripts for local startup.

Status: In progress — not yet completed.
